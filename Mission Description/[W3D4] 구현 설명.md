# [W3D4] Mission

고객 블랙리스트 명단을 작성해본다.

- `customer_blacklist.csv`파일을 만들고 Spring Application에서 해당 파일을 읽을 수 있고 블랙리스트를 조회할 수 있도록 설계한다.

- 바우처를 메모리에서 관리하는 Repository는 **개발 profile에서만 동작**하게 한다.
